Fira Code | 为写程序而生的字体

发表于2021-09-19|更新于2021-11-03|仓库字体
|字数总计:150|阅读时长:1分钟|阅读量:

安装

安装 Fira Code 编程字体

Github：https://github.com/tonsky/FiraCode/releases

Gitee：https://gitee.com/lovehuating/tool/blob/master/Fira_Code_v5.2_2.zip

安装这些 .ttf 后缀的文件

使用

打开vscode编辑器，左上角 文件 > 首选项 > 设置

点击 setting.json 编辑

默认 editor.fontLigatures 是 false，改为 true

然后添加字体选择

"editor.fontFamily": "'Fira Code', Consolas, 'Courier New', monospace"

PS：可以你设置了没有生效字体，你可以重启一下电脑，重启能解决99%的问题
