keep-alive 遇见 vue-router

keep-alive 是 Vue 内置的一个组件，可以是被包含的组件保留状态，或避免重新渲染

  • include - 字符串或正则表达，只有匹配的组件会被缓存

    Profile 和 User 都是组件的name属性值

  • exclude - 字符串或正则表达式，任何匹配的组件都不会被缓存

router-view 也是一个组件，如果直接被包含在 keep-alive 里面，所有路劲匹配到的视图组件都会被缓存

保留状态

生命周期函数 created destroyed

这样将不会重复执行created destroyed 函数


生命周期函数 activated deactivated

该生命周期只有当使用<keep-alive>嵌套<router-view> 才有用

当调用deactivated函数时，打印路径不是离开的页面，而是当前的页面。因为$route 没有来得及渲染，只能是当前页面的，故此方法不能保留状态
